# UI Clone: w.tho - Online Shop

 <a href="https://wtho-onlineshop.talismamanuel.repl.co">w.tho</a> is an online shop website, which a design with layout and typography experiments. made with just CSS HTML and javaScript. Credits to <a href="https://dribbble.com/muhsalmon">Muh Salmon</a> from dribble for layout ui design and my dear photographer friend <a href='https://www.instagram.com/samdiogophotography/'>Diogo Samuel</a> for his incredibles photos 
 

## Features

- css animatons
- css transitions
- Font Awesome using <i>
- hyperlink id
- DOM manipulation
- JavaScript class
- JavaScript Timing Events
- Responsive Web Design - Media Queries
